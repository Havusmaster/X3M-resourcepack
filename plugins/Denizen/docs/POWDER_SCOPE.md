# Система заряда пороха для Позорной трубы

## Как это работает

### Требования
- **Основная рука:** Подзорная труба (spyglass)
- **Вторая рука:** Порох (gunpowder)

### Механика выстрела

1. Игрок держит трубу в основной руке
2. Во второй руке должен быть порох
3. При **опускании трубы** (отпускание ПКМ) происходит выстрел
4. Количество пороха = сила выстрела

## Формула расчёта

```
Скорость = Количество пороха × 1.5
```

### Таблица зарядов

| Порох | Скорость | Описание |
|-------|----------|----------|
| 1 шт. | 1.5 | Очень слабый выстрел |
| 2 шт. | 3.0 | Слабый выстрел |
| 3 шт. | 4.5 | Средний выстрел |
| 4 шт. | 6.0 | Сильный выстрел |
| 5 шт. | 7.5 | Очень сильный выстрел |
| 6 шт. | 8.0 | Максимальный выстрел |
| 7 шт. | 8.0 | Максимальный (лимит)  будет тратится 6 а патом остнется 1 | 
| 8 шт. | 8.0 | Максимальный (лимит)  будет тратится 6 а патом остнется 1 | 

### Потребление пороха

```
Потрачено = Скорость / 1.2
```

Примеры:
- 1 порох → тратится ~0.8 ≈ 1
- 2 пороха → тратится ~1.6 ≈ 2 шт.
- 3 пороха → тратится ~3.5 ≈ 3 шт.
- 4 пороха → тратится ~4.3 ≈ 4 шт.
- 5 порохов → тратится ~5.2 ≈ 5 шт.
- 6 порохов → тратится 6 шт.

### Звук и эффекты

- **Звук:** Тем выше pitch, чем больше скорость
- **Эффект:** Дым при выстреле
- **Сообщение:** Показывает силу выстрела в чате

## Ограничения

- **Максимальная скорость:** 8 (больше не даёт эффекта)
- **Минимальный порох:** 1 шт. (без пороха - ошибка)
